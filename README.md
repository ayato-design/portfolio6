# SNSリンク集

## 概要
趣味で作成したページをポートフォリオ作品に再編集しました。
これまでの作品と同様に、HTML/CSS/JavaScriptを用いて作成しています。
レスポンシブ対応を行い、スマホでも閲覧し易いよう調整を行いました。

## 制作目的
- Figmaの操作練習の為
- HTML/CSSでのレイアウト構築に慣れる為
- JavaScript/jQueryを使った動きを実装する為
- 実務でありそうなレスポンシブ対応を意識する為

## 使用技術
- HTML
- CSS
- JavaScript(basic)
- jQuery
- Figma
- Canva

## 担当範囲
- デザインカンプを元にコーディング
- レスポンシブ対応
- スムーススクロール

## 工夫した点
- リンクのホバーアニメーションの実装。
- テキストリンクの判定拡張。
- PCとレスポンシブ時のグリット数の表記数変更。

## 苦労した点・解決方法
- 小さな凡ミスが多く、エラーが多数あった。

## 今後改善したい点
- 無駄にCSSを指定してしまっていて、流用(再利用)出来る部分は再利用したいです(余白だけ追加指定にするなど)
